This is a test example for this submission tool.

When you submit your solution it will executed against some test cases to evaluate if your solution is correct.

Some test cases are available to download in the end of the page. Some cases might not be available.

To submit your solution, click on the submit link and send your file.

In this scenario the solution is given in the end of the page.

In order to test your solution run it using the test cases.

To run it in your machine:

Download all the files to the same directory.

Execute the R script, enter one input file, check if it matches the output in the solution.

The judge script will run using:

$ cat | Rscript sample_submission.R >

$ diff

In this problem, create a script in R that will get in its input a filename. Open the dataset with the name given, then cat the filename in a line, then cat the mean value of the 'value' column in another line. Check the solution if needed.
